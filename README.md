# Rentkart 🏠⚡🚗

Rentkart is a full-stack rental e-commerce platform that allows users to rent essential appliances on a subscription basis.

## 🚀 Tech Stack
- Backend: Django, Django REST Framework
- Frontend: Angular (Standalone APIs), Bootstrap
- Authentication: OTP via Telegram Bot + JWT
- Database: SQLite (dev)
- Version Control: Git, GitHub

## 🔐 Features
- OTP-based admin login using Telegram
- JWT-secured APIs
- Role-based access (Admin / Customer)
- Category & Product management
- Subscription-based rental model
